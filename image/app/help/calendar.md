# Calendar — Help

The central calendar aggregates vacations, appointments and public holidays in a single monthly view.

## Navigating the calendar

Use **‹** and **›** to switch months. The current month is shown by default.

Each day shows:
- **Holidays** — public holidays for the configured state (Settings → Vacation → State)
- **Vacation entries** — approved entries in solid colour, planned/requested in dashed outline
- **Appointments** — training, conferences, team events and business trips
- **Task deadlines** — tasks with a due date on that day (✅ icon, red for high priority)

## Sprint flag

If sprints are configured, the first day of each sprint is marked with a flag icon (⚑).

## Day details

The list below the calendar shows all events for the selected month: holidays, vacation entries with status and note, and appointments with type and title.

## Sprint capacity

Click **Capacity** in the calendar header to open the per-sprint capacity overview showing available, blocked and remaining working days.

## Settings

- **State / region** — controls which public holidays are shown (Settings → Vacation)
- **Sprint configuration** — anchor date, duration and name prefix (Settings → Sprints)
- **Appointment types counted as blocked** — configurable in Settings → Sprints
