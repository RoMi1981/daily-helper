# Wissensdatenbank — Hilfe

Strukturierte Markdown-Einträge in Kategorien, gespeichert in einem Git-Repository.

## Eintrag erstellen

1. **+ Neuer Eintrag** in der Seitenleiste anklicken oder `n` drücken
2. **Kategorie** wählen (oder neu erstellen — keine Schrägstriche erlaubt)
3. **Titel** eingeben — wird zum Dateinamen
4. **Inhalt** in Markdown über die Toolbar schreiben
5. **Speichern & Pushen** klicken

## Bearbeiten

Eintrag öffnen und **Bearbeiten** klicken, oder auf der Detailseite `e` drücken.  
Bei Titeländerung wird die Datei umbenannt; bestehende Links werden automatisch weitergeleitet.

## Anheften

Auf einer Detailseite **☆ Anheften** klicken, um den Eintrag zu pinnen. Angeheftete Einträge erscheinen oben im Home-Dashboard. **★ Angeheftet** klicken zum Lösen.

## Suchen

Suchleiste nutzen (`/` zum Fokussieren) oder das Kategoriesuchfeld im Wissensdatenbank-Index.

## Tastaturkürzel

| Taste | Aktion |
|-------|--------|
| `/` | Globale Suche fokussieren |
| `n` | Neuer Eintrag |
| `e` | Bearbeiten (auf der Detailseite) |

## Markdown-Tipps

- `**fett**`, `*kursiv*`, `~~durchgestrichen~~`
- ` ```python ` … ` ``` ` — Code-Block mit Syntaxhervorhebung
- `- [ ]` — Checkbox-Liste
- `| Spalte | Spalte |` — Tabelle
