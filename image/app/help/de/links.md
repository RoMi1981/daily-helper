# Links — Hilfe

Lesezeichen nach Kategorie organisiert mit Volltextsuche und Einklick-Kopieren.

## Link hinzufügen

**+ Link hinzufügen** anklicken und ausfüllen:

| Feld | Hinweis |
|------|---------|
| **Titel** | Pflichtfeld |
| **URL** | Pflichtfeld, muss `https://` enthalten |
| **Kategorie** | Freitext; Autocomplete aus bestehenden Kategorien |
| **Beschreibung** | Optionaler Kurztext unterhalb des Titels |

## Durchsuchen

Links sind nach Kategorie gruppiert. Auf ein Kategorie-Badge klicken, um die Liste zu filtern.

## URL kopieren

Das **Kopieren**-Symbol auf einer Link-Karte anklicken, um die URL in die Zwischenablage zu kopieren.

## Link-Bereiche

Unter **Einstellungen → Link-Bereiche** können benannte Gruppen konfiguriert werden (z. B. pro Projekt oder Team). Jeder Bereich kann einen eigenen Floccus-kompatiblen API-Endpunkt haben.
