# Aufgaben — Hilfe

Aufgabenverwaltung mit Prioritäten, Fälligkeitsdaten und wiederkehrenden Aufgaben.

## Aufgabe erstellen

Das **Schnell-Hinzufügen**-Formular oben in der Aufgabenliste (Titel + optionales Fälligkeitsdatum) nutzen oder auf **+ Neue Aufgabe** klicken.

## Felder

| Feld | Hinweis |
|------|---------|
| **Titel** | Pflichtfeld |
| **Beschreibung** | Optionaler Text, der auf der Karte angezeigt wird |
| **Fälligkeitsdatum** | Optional. Überfällige Aufgaben werden rot markiert |
| **Priorität** | Hoch / Mittel / Niedrig — beeinflusst die Sortierung |
| **Wiederkehrend** | Täglich / Wöchentlich / Monatlich — erstellt beim Abschließen automatisch die nächste Aufgabe |

## Aufgabe abschließen

Checkbox auf der Aufgabenkarte anklicken. Die Aufgabe wechselt in den Bereich **Erledigt**.  
Bei wiederkehrenden Aufgaben wird automatisch die nächste Aufgabe mit dem nächsten Fälligkeitsdatum erstellt.

## Suchen

Das Suchfeld oben in der Aufgabenliste nutzen, um nach Titel oder Beschreibung zu filtern.
