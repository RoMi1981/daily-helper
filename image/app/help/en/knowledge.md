# Knowledge Base — Help

Structured Markdown entries organised in categories, stored in a git repository.

## Creating an entry

1. Click **+ New Entry** in the sidebar or press `n`
2. Select a **Category** (or create one — no slashes allowed)
3. Enter a **Title** — becomes the filename
4. Write **Content** in Markdown using the toolbar
5. Click **Save & Push**

## Editing

Open an entry and click **Edit**, or press `e` on the detail page.  
Changing the title renames the file; existing links redirect automatically.

## Pinning

Click **☆ Pin** on a detail page to pin an entry. Pinned entries appear at the top of the home dashboard. Click **★ Pinned** to unpin.

## Searching

Use the search bar (press `/` to focus) or the category search box on the Knowledge index.

## Keyboard shortcuts

| Key | Action |
|-----|--------|
| `/` | Focus global search |
| `n` | New entry |
| `e` | Edit (on detail page) |

## Markdown tips

- `**bold**`, `*italic*`, `~~strikethrough~~`
- ` ```python ` … ` ``` ` — fenced code block with syntax highlighting
- `- [ ]` — checkbox list
- `| col | col |` — table
