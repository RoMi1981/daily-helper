# Links — Help

Bookmarks organised by category with full-text search and one-click copy.

## Adding a link

Click **+ Add Link** and fill in:

| Field | Notes |
|-------|-------|
| **Title** | Required |
| **URL** | Required, must include `https://` |
| **Category** | Free text; autocomplete from existing categories |
| **Description** | Optional short text shown below title |

## Browsing

Links are grouped by category. Click a category badge to filter the list.

## Copying a URL

Click the **copy** icon on a link card to copy the URL to clipboard.

## Link Sections

Configure **Link Sections** in **Settings → Link Sections** to organise bookmarks into named groups (e.g. per project or team). Each section can have its own Floccus-compatible API endpoint.
