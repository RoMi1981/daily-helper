# Snippets — Help

Reusable command snippets with per-command copy buttons and full-text search.

## Creating a snippet

1. Click **+ New Snippet**
2. Enter a **Title** (required) and optional **Description**
3. Add commands with **+ Add Command** — each command has an optional description and the command/code itself
4. Reorder commands with the ↑ ↓ buttons

## Using a snippet

Open a snippet to see all commands. Click **Copy** next to any command to copy it to clipboard.

## Searching

Use the search box to filter by title, description or command content.
