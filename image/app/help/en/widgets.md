# Widget Dashboard

The **Widget Dashboard** (`/widgets`) is a configurable home screen. Choose which widgets to display and arrange them freely via drag-and-drop.

## Edit Mode

Click the **Edit** button (pencil icon) in the top right to enter edit mode:

- **Drag** widgets by the ⠿ handle to reorder them
- **Remove** individual widgets with the ✕ button
- **Remove Selected** — check multiple active widgets, then click to remove them all at once
- **Remove All** — removes all widgets from the dashboard
- Click **Done** to save the new layout

## Adding Widgets

In edit mode, the **Add Widget** panel appears below the grid:

- Click a widget button to add it immediately
- Check multiple widgets, then click **Add Selected**
- Click **Add All** to add every available widget

## Widget Types

**Stats widgets** (⅓ width) show the entry count for a module with a link to that module. Two stats widgets fit side by side on mobile.

**Content widgets** show live data — upcoming appointments, overdue tasks, sprint countdown, RSS feed, recent notes, favorites, and more.

## Widget Settings

Some widgets have a settings gear icon (⚙) in edit mode. Click it to configure options such as the number of items shown or which RSS feed to display.

## Layout Persistence

The layout is saved per user in Settings. Drag order and enabled/disabled state are preserved across page reloads.
