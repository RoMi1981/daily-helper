# Tasks — Help

Task management with priorities, due dates and recurring tasks.

## Creating a task

Use the **Quick-Add** form at the top of the task list (title + optional due date), or click **+ New Task** for the full form.

## Fields

| Field | Notes |
|-------|-------|
| **Title** | Required |
| **Description** | Optional free text shown on the card |
| **Due date** | Optional. Overdue tasks are highlighted in red |
| **Priority** | High / Medium / Low — affects sort order |
| **Recurring** | Daily / Weekly / Monthly — creates next task automatically on completion |

## Completing a task

Click the checkbox on the task card. The task moves to the **Done** section.  
For recurring tasks a new task is created with the next due date.

## Searching

Use the search box at the top of the task list to filter by title or description.
