# Vacations — Help

Vacation request management with automatic day calculation and account tracking.

## Creating a request

Click **+ New Request** and select **From** and **To** dates.  
Weekends and public holidays (configured in **Settings → Vacation**) are excluded from the day count automatically.

## Status workflow

| Status | Meaning |
|--------|---------|
| **Planned** | Draft — not yet submitted |
| **Requested** | Submitted for approval |
| **Approved** | Confirmed |
| **Rejected** | Declined |

Use the status buttons on each request card to advance the workflow.

## Account overview

The **Account** panel shows total days, used days and remaining balance for the current year, including optional carryover from the previous year.

## Settings

Go to **Settings → Vacation** to configure:
- **Federal state** — determines which public holidays are excluded
- **Days per year** — annual entitlement
- **Carryover** — days carried over from the previous year
- **Vacation mail template** — pre-fills the vacation notification email
